## 🍥 노트
### 1. 시간 복잡도
간선을 탐색하는 시간: O(E)<br/>
근데 간선을 탐색하면서 우선순위 큐에 원소를 삽입: O(logN)<br/>
따라서 최단거리 구하는 과정의 시간복잡도: O(ElogN)
전체적인 시간복잡도: 인접 리스트를 만드는 과정과 합치면 O(E+ElogN)=O(ElogN)
### 2. 개선점
