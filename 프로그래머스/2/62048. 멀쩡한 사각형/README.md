## 노트
### 모든 원흉은 자료형 변환에
우선 이 문제에서는 h/w*i -> h*i/w 로 바꾸고 통과했는데, 자연수끼리 곱셈 연산을 한 후 나누기를 수행하면 오차가 줄기 때문에 통과했음!<br/>
그리고 h(<=10억)와 w(<=10억)을 곱할 때, 계산 결과를 long long에 할당한다고 해도 계산 과정에서 오버플로우가 날 수 있기 때문에<br/>
(long long)h*w로 형변환 해야 함!
